# gigawallet-docs

Documentation for GigaWallet, available at gigawallet.dogecoin.org

Built with [Fumadocs](https://fumadocs.dev/) (Next.js + MDX).

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
pnpm build
```

Static output is in the `out/` directory (for GitHub Pages deploy).

## Deploy (GitHub Pages)

The repo uses GitHub Actions to build and deploy to GitHub Pages. Ensure **Pages** is set to deploy from the **GitHub Actions** source in your repository settings.
